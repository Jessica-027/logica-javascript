# Desafio 004 - Sistema de Descontos em Compras

Projeto desenvolvido para praticar lógica de programação com JavaScript.

## Objetivos

- Tratamento de entradas
- Validação de dados
- Conversão de tipos
- Regras de negócio
- Cálculo de descontos
- Organização em funções
- Separação de responsabilidades

## Regras

- Todos os campos obrigatórios devem ser preenchidos.
- O usuário pode cancelar a operação.
- A idade e o valor da compra devem ser números válidos.
- A idade não pode ser negativa.
- O valor da compra deve ser maior que zero.
- As formas de pagamento aceitas são DINHEIRO, PIX e CARTÃO.
- Clientes com 60 anos ou mais recebem desconto.
- Pagamentos em dinheiro ou PIX recebem desconto.
- Compras com valores maiores recebem descontos progressivos.
- O desconto total é limitado a 20%.
- O valor final da compra nunca pode ser menor que R$ 10,00.

- Funções

- Objetos
- Tratamento de dados

- Conversão de valores
- Regras condicionais
