# Desafio 003 - Calculadora de Frete

Projeto desenvolvido para praticar lógica de programação com JavaScript.

## Objetivos

- Tratamento de entradas
- Validação de dados
- Conversão de tipos
- Regras de negócio
- Organização em funções
- Separação de responsabilidades

## Regras

- A taxa inicial do frete é R$ 10,00.
- O valor do frete depende do peso da encomenda.
- O valor do frete depende da distância da entrega.
- O tipo de entrega altera o valor final.
- Produtos acima de R$ 500,00 recebem desconto no frete.
- O frete nunca pode ter valor negativo.

## Aprendizados

- Funções
- Parâmetros
- Objetos
- Tratamento de dados
- Validações
- Conversão de valores
- Fluxo de execução
- Organização de código
