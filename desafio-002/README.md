# Sistema de Análise de Empréstimo

Projeto desenvolvido para praticar lógica de programação com JavaScript.

## Objetivos

- Validação de entradas
- Conversão de dados
- Tratamento de erros
- Regras de negócio
- Organização em funções
- Separação de responsabilidades

## Regras

- O usuário deve ter 18 anos ou mais.
- Salário deve ser maior que zero.
- Valor do empréstimo deve ser maior que zero.
- O empréstimo não pode ultrapassar 10 vezes o salário mensal.

## Tecnologias

- JavaScript
